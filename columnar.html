Level 5: Columnar Transposition

This is a columnar transposition cipher. The plaintext is written in rows, then the columns are read off in a different order based on a keyword.

Hint: The keyword used was CIPHER. Remove spaces from the message, write it left-to-right under the keyword, then read the columns in alphabetical order of the letters in CIPHER.
